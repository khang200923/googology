Psi array:


\(\psi\) array uses the superpower of the \(\psi\) function to boost the number extremely and uncomputably fast!


Definition:

Note: \(\#\) is defined as the portion of the array; \(()^*_n\) shows the part of the array, where \(n\) is the base of the array (if \(()^*\), means it does not need the base).

This is \(\psi\{a\}_n\) where \(a\) (Array) is a sequence consists of countable ordinals and \(n\) (Base) is a nonnegative integer.

Basic rules

Output: \(\psi\{\}_n = n\)
Empty: \((\#_1{0}\#_2)^* = (\#_1)^*\)
Psi-ing: \((n,m) = (n-1,\psi(m))\) where \(n\) and \(m\) are two different arguments in the array or the base.
Wainering: \((n,m) = (n,m[n])\) where \(n\) and \(m\) are two different arguments in the array or the base.
Least prior rules are Psi-ing and Wainering, used if none of the rules are applied. If not unique, pick 2 numbers that apply to the rule to perform that results in a big number.

Linear array rules

Recursion: \(\psi\{\#a,b\}_n = \psi\{\#\psi\{\#a,b-1\}_n,b-1\}_{\psi\{\#a,b-1\}_n}\)

Dimension array rules

Comma: \(({\#_1},{\#_2})^* = ({\#_1}[0]{\#_2})^*\) Recursion: \((a[c]b\#)^*_n = (a[c-1]a[c-1]a[c-1]a...[c]b-1)^*_n\) where there are \(n\) \(a\)'s.
