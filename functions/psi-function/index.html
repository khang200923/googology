Psi function:

This \(\psi\) notation is an uncomputable function, and one of the googological functions that uses real number as the input.

Definition

First, define \(\alpha_n\) where \(n\) is a nonnegative integer as follows:

\(\alpha_0 = \omega\)
\(\alpha_1 = \omega^2\)
\(g_{\alpha_n}(2) = 4\) where \(g\) is SGH based on Veblen hierarchy.

Define \(q(n) = 1 - 1 / n\).

Then, define \([x]\) where \(x\) is a finite sequence with nonnegative integers.

Here, \(x_m\) describes any nonegative numbers, \(y_m\) represents the rest of the notation (can be empty):

\([x_1] = q(x_1)\)
\([y_1, x_1, x_2] = [y_1, x_1] + ([y_1, x_1+1] - [y_1, x_1]) * q(x_2)\)
\([y_1] = [y_1, 1]\)


Every countable ordinal \(\beta\) can be represented as \(\beta = \alpha_n[a_1][a_2][a_3][a_4]...\) with respect to Veblen Hierarchy. This form is unique, so define \(w(\beta)\) for countable ordinal \(\beta\) as \(n + [a_1,a_2,a_3,a_4,...,a_x] - 1\) satifies \(\beta = \alpha_n[a_1][a_2][a_3][a_4]...[a_x]\).

\(T = \{x+y|\beta, y=w(\beta)\}\) where \(\beta\) is a countable ordinal and x is a nonnegative ordinal.

Finally (n is a nonnegative real number):

\(\psi(n) = \psi(m)\) where \(m\) is the nearest value from set \(T\) of \(n\)
\(\psi(n) = f_{psi(n)}(10)\) (FGH with Wainer hierarchy) where \(psi(n)\) (a completely different function) satisfies \(w(psi(n)) = n\)
